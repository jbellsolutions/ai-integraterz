## Your Claude Code Setup

Install Claude Code: https://claude.ai/code

Once installed, copy the CLAUDE.md file from your delivery package into your home directory:
```bash
cp CLAUDE-Owner.md ~/.claude/CLAUDE.md
```

This configures Claude Code to know exactly who you are and how you work.

## Your Daily AI Routine

**Morning (15 minutes):**
- Open Claude Code
- Type: "Give me my morning brief" — Claude will pull your priorities, emails needing responses, and key decisions for the day

**Throughout the day:**
- When writing anything: "Help me write [X] in my voice"
- When making a decision: "Think through [decision] with me — here's the context"
- When in a meeting: After it ends, paste your notes and type "Clean this up and pull out action items"

**End of day (10 minutes):**
- Type: "Summarize what I accomplished today and what's pending for tomorrow"
