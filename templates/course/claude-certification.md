# Claude Code Certification
## AI Integraterz Official Certification Program

---

## What This Certification Means

Earning the AI Integraterz Claude Code Certification means you can use Claude Code to do your actual job — not hypothetical examples, not generic prompts, but your real work, faster and better.

It is not a test of AI theory. It is a demonstration of practical skill.

---

## Certification Requirements

To earn certification, complete all four modules and the practical assessment:

| Module | Topic | Time |
|--------|-------|------|
| Module 1 | Claude Code Fundamentals | 2 hours |
| Module 2 | Role-Specific Workflows | 3 hours |
| Module 3 | Advanced Prompting | 2 hours |
| Module 4 | Building Your Own Workflows | 2 hours |
| Assessment | Practical Demonstration | 1 hour |

**Total time:** ~10 hours (spread over 2-4 weeks with coaching support)

## Module 1: Claude Code Fundamentals

### 1.1 What Claude Code Is (and Isn't)
- Claude Code vs. ChatGPT vs. Claude.ai — the differences that matter
- Why business users need Claude Code specifically
- What "AI-native" actually looks like in practice

### 1.2 Setup
- Installing Claude Code on Mac/Windows
- Copying your CLAUDE.md configuration (your personal AI profile)
- Understanding the slash command system
- Your first prompt: "Help me with [your most common task]"

### 1.3 The Basics That Actually Matter
- How to give Claude the right context
- When to start fresh vs. continue a conversation
- Reading output critically (the review habit)
- The "Colleague Test" — does it sound like a human or a robot?

**Module 1 Exercise:** Complete 3 of your real work tasks using Claude Code. Document the prompts you used and the time saved.

## Module 3: Advanced Prompting

### 3.1 Context Is Everything
- How to give Claude exactly what it needs
- Pasting the right data (emails, reports, notes, meeting transcripts)
- The "before you answer" technique
- Iterating on output vs. starting over

### 3.2 Voice Matching
- Getting Claude to write in your voice
- The voice profile prompt
- When to edit vs. regenerate
- The Colleague Test in practice

### 3.3 Multi-Step Tasks
- Breaking complex tasks into steps
- Using Claude to plan before executing
- The "think through this with me" prompt
- When to use Claude Code vs. Claude.ai

### 3.4 Common Mistakes and How to Fix Them
- "I got a generic answer" → Add more context
- "It's too long" → Tell Claude the format you want first
- "It doesn't sound like me" → Use your voice profile
- "It made something up" → Ask it to cite its sources or flag uncertainty

**Module 3 Exercise:** Take one task that took you >30 minutes last week. Complete it using Claude Code in under 10 minutes. Document your prompt sequence.

## Practical Assessment

To complete certification, demonstrate ONE of the following in a live coaching session:

**Option A: Speed Demonstration**
Complete a task that normally takes you 1 hour in under 15 minutes using Claude Code.

**Option B: Workflow Presentation**
Walk through 3 of your core workflows, explain the prompt you use for each, and show the output you get.

**Option C: New Workflow Build**
On the call, take a real problem from your current work and build a new AI workflow for it from scratch.
